# 贡献指南

感谢您对BNBU-FM课程攻略项目的关注和支持！我们欢迎所有BNBU的同学和校友贡献课程资料，共同完善这个共享平台。

## 如何贡献

### 1. 准备贡献的资料

您可以贡献以下类型的资料：

- 📚 **课件幻灯片** - 教师提供的PPT或PDF课件
- 📝 **作业与答案** - 课程作业、练习题及参考答案
- 📖 **笔记与总结** - 学习笔记、知识点总结
- 📊 **考试资料** - 期中/期末考试试卷、模拟题
- 📈 **补充材料** - 参考书、学习指南、实验报告等

### 2. 资料要求

为确保资料质量，请遵循以下要求：

- 确保内容清晰可读
- 避免包含个人信息
- 尊重知识产权，仅分享允许共享的资料

### 3. 提交方式

#### 方式一：GitHub Issue

- 如果您熟悉Git和GitHub，您可以在issue中许愿，项目管理员会尽量收集您所需课程的资料
- 如果您对某些课程由独特的经验和心得，也可以通过这个方式分享给我们

#### 方式二：邮件提交

1. 将资料整理打包
2. 发送邮件至：fm-courser@outlook.com
3. 邮件主题格式：`[课程资料贡献] 课程名称`
4. 在邮件中说明资料内容和来源

## 贡献流程

### 1. 确认课程

首先确认您要贡献的课程是否已在平台中：

1. 查看[课程列表](/courses/)确认课程是否存在
2. 如果课程不存在，请先联系我们添加新课程

### 2. 整理资料

按以下步骤整理资料：

1. **分类整理**：按文件类型（课件、作业、考试等）分类
2. **重命名文件**：按照命名规范重命名文件
3. **检查内容**：确保文件内容完整、清晰
4. **压缩打包**：将相关文件打包为ZIP格式

### 3. 提交资料

根据您选择的提交方式提交资料。

## 贡献者权益

所有贡献者将被记录在[致谢页面](/thanks/)中。
